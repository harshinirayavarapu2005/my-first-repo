DevOps Lifecycle

Plan-Defines business requirement.
⬇
Develop-Write and manage source code collaboratively.
⬇
Build-Compile code and build artifacts.
⬇
Test-Automatically test the code and fix the bugs.
⬇
Release-Approve and package the build for deployment.
⬇
Deploy-Push the code to production.
⬇
Operate-Manage the application in production.
⬇
Monitor-Continously monitor performance,logs etc..,
